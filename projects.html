Portfolio Site Project

I needed a portfolio and resume site for my future in software development. So it being Winter break at school i decided this was a great time to build a site for this purpose. The site is pretty simple over all but my goal was to build a fully functioning site to display my brand from scratch. I used Bootstrap for simplicity and will be adding more web technologies to the site over time. With term two around the corner time may be scarce. Hope you like it.

String Encryption Project

I did this as part of a group project. Me and my team built software that did simple string obfuscation using a substitution cipher. The software itself is very simplistic however, it was simply an introduction to encryption in general. I found it very interesting and will most likely be looking into doing more in this area.
